Spectral DB search

Description

Run on any feature list to match all rows agains a local spectral library (formats: MoNA json, NIST msp, GNPS json (internal library submission format)).

Method parameters

Masslist
Masslist of the MS2 scans
Spectral database file
json/msp from MoNA (MassBank of North America)
msp NIST format
json GNPS (format from the spectral DB submission module)
Retention time tolerance
Optional: Only used if the DB entry has a retention time
Noise level
An additional noise level (masslists are typically already thresholded)
Min cos similarity
Minimum cosine similarity between fragmentation scan and DB entry
Minimum matched signals
Minimum number of signals that need to match the m/z tolerance
