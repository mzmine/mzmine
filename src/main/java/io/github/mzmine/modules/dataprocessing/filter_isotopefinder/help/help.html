Isotope finder

Description

This module searches for possible isotope signals in MS1 data in the FWHM of all features. Compared to the isotope filter (useful to remove isotope features from the feature list), this module aims to provide a complete set of possible isotope signals with they mean relative height across multiple scans. However, not all signals will be isotope signals.

The most appropriate place to apply this module might be after feature deconvolution and deisotoping (if applied).

Method parameters

Name suffix
Suffix to be added to peak list name
m/z tolerance
Maximum distance in m/z from the expected location of a peak
Elements
Elements to consider for the isotope pattern
Maximum charge
Maximum charge to consider for detecting the isotope patterns
