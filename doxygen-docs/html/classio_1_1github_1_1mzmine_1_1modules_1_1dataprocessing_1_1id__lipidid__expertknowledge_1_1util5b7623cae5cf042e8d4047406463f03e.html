Detailed Description

Class that represents the MobilePhases in the setup dialog for the user to choose. The user can choose none, or all of them at the same time. The main logic was derived form similar classes in other modules, and the checkboxes were done with the help of ChatGPT.

Creates a new MobilePhaseComponent. It specifies the size of the window, buttons "All", choose all the options at once, and "Clear" to clear all options, and the way the are displayed. The user can also, instead of using the buttons click on the checkboxes individually.
